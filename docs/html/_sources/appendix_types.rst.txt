Appendix B: Data Types
=======================

The SBML attribute types recognized by Deviser Edit and the corresponding C++ data
type.

+-----------------------------------+-----------------------------------+
| **Deviser Edit attribute Type**   | **Corresponding C++ data type**   |
+===================================+===================================+
| string                            | std::string                       |
+-----------------------------------+-----------------------------------+
| boolean                           | bool                              |
+-----------------------------------+-----------------------------------+
| double                            | double                            |
+-----------------------------------+-----------------------------------+
| integer                           | int                               |
+-----------------------------------+-----------------------------------+
| unsigned integer                  | unsigned int                      |
+-----------------------------------+-----------------------------------+
| positive integer                  | unsigned int                      |
+-----------------------------------+-----------------------------------+
| non-negative integer              | unsigned int                      |
+-----------------------------------+-----------------------------------+
| ID                                | std::string                       |
+-----------------------------------+-----------------------------------+
| IDREF                             | std::string                       |
+-----------------------------------+-----------------------------------+
| SId                               | std::string                       |
+-----------------------------------+-----------------------------------+
| SIdRef                            | std::string                       |
+-----------------------------------+-----------------------------------+
| UnitSId                           | std::string                       |
+-----------------------------------+-----------------------------------+
| UnitSIdRef                        | std::string                       |
+-----------------------------------+-----------------------------------+
